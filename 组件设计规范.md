# 低空智能巡检平台 - 组件设计规范

本文档根据「底空智能巡检平台」组件规范设计文件整理，定义了平台的基础配色、字体、图标及通用组件的视觉与交互规范。

## 1. 配色规范 (Color Palette)

平台采用深色科技感风格，以深蓝色为基调，亮蓝色作为主要品牌交互色。

### 1.1 核心色彩

| 类别         | 变量名/属性       | 颜色值    | 示例 | 应用描述                         |
| :----------- | :---------------- | :-------- | :--- | :------------------------------- |
| **主色**     | Background (Base) | `#24457A` |      | 整体大背景色，深蓝色调           |
| **品牌色**   | Primary (Active)  | `#0ABAFF` |      | 亮蓝色，用于高亮、按钮、激活状态 |
| **背景色**   | Table Row (Alt)   | `#0F1F3A` |      | 列表/表格数据行背景色            |
| **表头背景** | Table Header      | `#172C51` |      | 列表/表格表头背景色              |

### 1.2 辅助色彩

| 颜色   | 颜色值    | 说明                       |
| :----- | :-------- | :------------------------- |
| 浅青色 | `#51F4F3` | 辅助强调色、数据图表色     |
| 橙色   | `#FC6533` | 辅助强调色、数据图表色     |
| 黄色   | `#FBA625` | 辅助强调色、警告类文字背景 |
| 紫色   | `#AD5DFF` | 辅助强调色                 |

### 1.3 状态/提示色

| 状态           | 颜色值    | 说明                         |
| :------------- | :-------- | :--------------------------- |
| 成功 (Success) | `#19BE6B` | 成功提示、正向数据状态       |
| 错误 (Error)   | `#FF6666` | 错误提示、异常状态、删除按钮 |
| 警告 (Warning) | `#FFC43A` | 警告提示、待处理状态         |
| 提示 (Info)    | `#6C76F4` | 普通操作提示、系统信息       |

## 3. UI 组件规范 (Common Components)

### 3.1 按钮 (Buttons)

- **确定/登录按钮**：`#0ABAFF` 渐变背景（亮蓝色），白色文字，圆角设计。
- **功能按钮 (新增等)**：`#0ABAFF` 蓝色描边，深色半透明背景，蓝色文字。
- **次要按钮 (重置等)**：深色描边，白色或淡灰色文字。
- **危险按钮 (删除等)**：`#FF6666` 红色文字。

### 3.2 列表与表格 (Lists & Tables)

- **表头**：`#172C51` 背景，文字颜色通常为浅蓝色或亮蓝色。
- **数据行**：`#0F1F3A` 背景，行间有暗色分割线。
- **状态标识**：任务状态采用不同颜色区分，如：已完成（`#19BE6B`）、进行中（`#0ABAFF`）。

### 3.3 弹窗 (Modals/Dialogs)

- **背景**：深蓝色半透明遮罩。
- **标题栏**：左侧带有 `#0ABAFF` 亮蓝色垂直装饰条。
- **操作区**：按钮通常位于底部右侧，遵循「确定」在右，「取消」在左的布局。

### 3.4 表单元素 (Form Items)

- **输入框/下拉框**：深色背景，`1px` 亮蓝色边框。
- **Label**：默认使用 `#0ABAFF` 亮蓝色文字，增强科技感。

---

## 4. 视觉标识 (Visual Icons)

- **装饰性标签**：
  - **六边形标签**：带有 `#00E3D2` 到 `#3F77E9` 的青蓝渐变边框，用于大屏展示。
  - **胶囊竖签**：垂直排列的文字外围包裹圆角渐变边框。
- **图标风格**：均为线性（Outlined）科技风图标，颜色统一使用亮蓝色或渐变青色。
